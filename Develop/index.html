The Orphaned Starfish Foundation Redesign

Overview

The website was a little outdated, there was no clear brand palette which made the website less aesthetically pleasing. Through research and user testing we found that, users were frustrated with the lack of payment options for donations. To build and create an impactful homepage that evokes emotion to increase donations. To have a clear and transparency stance on how each donation impacts each orphan, as well as providing multiple payment options.

Research Phase

Through our user research and stakeholder interview we found that the stakeholders number one priority was to receive more donations, and users wanted to give more donations with more payment options. Did interview plans, interviews with users and the stakeholder, surveys.

Definition Phase

Graphed the user insight and empathy map as well as the affinity diagram. Worked on user/proto persona, key insight, main features, problem statement, value preposition, etc. Afterwards we did a user journey map, user flow and storyboard.

Ideation Phase

Reflected on the research and made an 'I Like, I Wish, What If" board with our most important findings. Formulated the user journey map and the user flow and started out the sketching process.

Prototyping Phase

Started prototyping our sketches, devided the plan up a few pages/person and then agreed upon our style choices such as colors, typography, button states etc. Chose to do a mobile version as well in order to make it more accessible and allow device variety.

Testing Phase

Did user testing at every stage of our project: Low-Fi, Mid-Fi, Hi-Fi and A/B testing. Reiterated every step and every stage of the prototype for accessibility and user intuitiveness.

Conclusion

Opportunities: implement more of the pages of the website, create more direct links when necessary,improve user flow overall, create an event calendar, create more/better opportunities for people to help, implement an API that calculates how much your donation actually helped and who it is allocated to, further the development of the website for a more smooth User Experience and better reach.
