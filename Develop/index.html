The Orphaned Starfish Foundation Redesign

Creating an impactful homepage that evokes emotion and increases donations.

Overview

The website was a little outdated, with no clear brand palette, making it less aesthetically pleasing. Through research and user testing, we found that users were frustrated with the lack of payment options for donations. The goal was to create an impactful homepage that evokes emotion and increases donations, providing clear transparency on how each donation impacts orphans and offering multiple payment options.

Research Phase

Through our user research and stakeholder interviews, we found that the stakeholders' number one priority was to receive more donations, and users wanted to give more donations with more payment options. We conducted interview plans, interviews with users and stakeholders, and surveys.

Definition Phase

We graphed user insights, created an empathy map, and developed an affinity diagram. We worked on user/proto personas, key insights, main features, problem statements, and value propositions. Afterward, we created a user journey map, user flow, and storyboard.

Ideation Phase

We reflected on the research and created an 'I Like, I Wish, What If' board with our most important findings. We formulated the user journey map and user flow, and started the sketching process.

Prototyping Phase

We started prototyping our sketches, dividing the plan into a few pages per person, and then agreed upon our style choices such as colors, typography, button states, etc. We also chose to create a mobile version to make it more accessible and allow device variety.

Testing Phase

We conducted user testing at every stage of our project: Low-Fi, Mid-Fi, Hi-Fi, and A/B testing. We reiterated at every step and stage of the prototype for accessibility and user intuitiveness.

Conclusion

Opportunities: implement more pages of the website, create more direct links when necessary, improve user flow overall, create an event calendar, create more/better opportunities for people to help, implement an API that calculates how much your donation actually helped and who it is allocated to, and further the development of the website for a smoother user experience and better reach.
